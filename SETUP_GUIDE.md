# FX Date Calculator 구동 가이드

## 1. 사전 준비

**Node.js 설치** (이미 설치되어 있다면 건너뛰세요)
- https://nodejs.org 접속
- LTS 버전 다운로드 후 설치

설치 확인:
```bash
node -v
npm -v
```

---

## 2. 실행 방법

### Step 1: 압축 풀기
`fx-date-calculator-web-v19.zip` 압축을 원하는 폴더에 풀어주세요.

### Step 2: 터미널 열기
- **Mac**: Terminal 앱 실행
- **Windows**: PowerShell 또는 CMD 실행

### Step 3: 폴더로 이동
```bash
cd 압축푼경로/fx-date-calculator-web
```

### Step 4: 패키지 설치 (최초 1회)
```bash
npm install
```

### Step 5: 실행
```bash
npm run dev
```

### Step 6: 브라우저에서 접속
```
http://localhost:3000
```

---

## 3. 종료 방법
터미널에서 `Ctrl + C`
